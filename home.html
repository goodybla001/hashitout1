Mesh Network App Detection- KVH-015

Design and develop a technological solution for detecting apps like fire chat that uses Mesh Networking to connect users without cell in a given area. The solution should be man-portable and should be able to scan an area corresponding to a relatable TSP-BTS(Telecommunications Service Provider-Base Transceiver Station).

About JaalNetr

Introducing Jaalnetra, a cutting-edge system designed to detect and analyze WiFi and Bluetooth mesh networks in real-time. With advanced signal processing, topology mapping, and traffic analysis, Jaalnetra provides a clear overview of network structure and performance metrics. Its robust security features include intrusion detection, encryption analysis, and anomaly alerts, ensuring optimal network efficiency and safeguarding against potential threats. Seamlessly integrating with existing tools, Jaalnetra empowers administrators to harness the full potential of mesh networks while maintaining a secure wireless communication environment.
